# CanonRS

Enterprise-grade UI component library for Leptos with SSR support.

## Crates

- **canonrs** - Facade (main entry point)
- **canonrs-shared** - Shared types and utilities
- **canonrs-ui** - SSR-safe UI components
- **canonrs-ui-interactive** - Interactive wrappers (CSR/hydrate)
- **canonrs-providers** - Context providers (SSR + hydrate)
- **canonrs-csr** - Client-side runtime

## Installation
```toml
[dependencies]
canonrs = "0.1"
```

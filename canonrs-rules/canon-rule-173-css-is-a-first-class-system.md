# Canon Rule #173: CSS Is a First-Class System

**Status:** ENFORCED  
**Severity:** HIGH  
**Scope:** architecture, css
**Version:** 1.0.0  
**Date:** 2026-01-28

---

## Principle

**CSS in CanonRS is a governed system, not decoration.**

---

## Problem

Treating CSS as secondary leads to untraceable UI bugs.

---

## Forbidden Pattern

❌ Ad-hoc CSS fixes without architectural grounding.

---

## Canonical Pattern

✅ CSS follows the same rigor as Rust code.

---

## Enforcement

- Rule compliance
- Design audits

---

## Exceptions

None.
